## ⚠️ 回滚方案

如果v2效果不理想，可以快速回滚到v1：

### 方案A: 切换版本（推荐）

```sql
-- 切换回v1
UPDATE labels
SET is_active = (version = 1)
WHERE name = '涉警当事人信息完整性检查';

-- 验证
SELECT id, version, is_active
FROM labels
WHERE name = '涉警当事人信息完整性检查';
```

### 方案B: 删除v2（彻底回滚）

```sql
-- 备份v2配置
SELECT * FROM labels
WHERE version = 2
INTO OUTFILE '/tmp/label_v2_backup.csv';

-- 删除v2
DELETE FROM labels
WHERE name = '涉警当事人信息完整性检查'
  AND version = 2;
```
