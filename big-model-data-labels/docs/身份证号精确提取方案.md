## 1. 问题分析

### 1.1 核心需求

从一段文字中提取**所有满足18位**的身份证号，同时确保：
- ✅ 准确提取18位身份证号
- ✅ 排除15位旧身份证号
- ✅ 避免将其他18位数字（订单号、序列号等）误识别
- ✅ 处理格式不规范（带空格、分隔符、大小写x等）

### 1.2 技术挑战

| 挑战 | 描述 | 传统方案的问题 |
|------|------|---------------|
| **区分15位和18位** | 文本中混有15位旧身份证号 | 纯正则无法区分，会误提取15位的部分数字 |
| **排除其他18位数字** | 订单号、序列号、随机数字 | 纯正则误提取率高（30-40%） |
| **格式不规范** | 带空格、横杠、大小写x | 需要标准化处理 |
| **上下文理解** | 判断数字是否确实为身份证号 | 纯规则无法理解语义 |

### 1.3 方案对比

| 方案 | 准确率 | 召回率 | 速度 | 成本 | 适用场景 |
|------|--------|--------|------|------|----------|
| **纯正则** | 60-70% | 95%+ | ⚡ 极快 | 💰 零成本 | 简单场景，可容忍误提取 |
| **纯大模型** | 95%+ | 95%+ | 🐌 较慢 | 💸💸💸 高成本 | 复杂场景，要求极高准确性 |
| **正则+格式验证** | 85-90% | 95%+ | ⚡ 快 | 💰 低成本 | 标准格式数据 |
| **四层混合方案** | **98%+** | **95%+** | ⚡ 较快 | 💰💸 中等成本 | **生产环境，要求高准确性** |

**推荐**：四层混合方案，在保证准确性的同时控制成本。

## 2. 解决方案设计

### 2.1 四层验证机制架构

```
输入文本
    ↓
【第一层】正则表达式初筛
    → 提取所有18位数字序列（包括带X结尾的）
    → 支持带分隔符的格式（空格、横杠、点）
    → 快速过滤明显不符合格式的数字
    ↓
【第二层】格式验证
    → 验证地区码合法性（前6位，查询行政区划数据库）
    → 验证出生日期合法性（第7-14位，1900-当前年份）
    → 验证校验位正确性（第18位，ISO 7064算法）
    → 过滤约60-70%的误候选
    ↓
【第三层】上下文语义验证（DeepSeek 70B）
    → 使用70B大模型理解上下文语义
    → 判断文本中是否有"身份证"相关关键词
    → 排除订单号、序列号、随机数字等
    → 对通过格式验证的候选进行最终确认
    ↓
【第四层】人工审核（可选）
    → 对低置信度结果（<80%）进行人工确认
    → 收集误提取案例，优化规则
    ↓
输出：高置信度的18位身份证号列表 + 详细验证日志
```

### 2.2 核心设计原则

1. **分层过滤**：每层过滤掉明显不符合条件的候选，减少下一层处理量
2. **快速失败**：先执行成本低的验证（正则、格式），最后调用大模型
3. **保守策略**：大模型验证失败时，如果格式验证全部通过，倾向于保留
4. **详细日志**：记录每层验证结果，便于审计和调试

## 4. 后端实现

### 4.1 身份证提取服务（核心）

#### `IdCardExtractionService.java`

```java
package com.datalabeling.service;

import com.datalabeling.dto.response.IdCardResult;
import com.datalabeling.dto.response.SemanticValidationResult;
import com.datalabeling.entity.Label;
import com.datalabeling.model.IdCardCandidate;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import lombok.Data;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.time.LocalDate;
import java.time.format.DateTimeFormatter;
import java.time.format.DateTimeParseException;
import java.util.*;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

@Slf4j
@Service
public class IdCardExtractionService {

    @Autowired
    private DeepSeekService deepSeekService;

    @Autowired
    private AreaCodeService areaCodeService;

    private final ObjectMapper objectMapper = new ObjectMapper();

    /**
     * 从文本中提取18位身份证号（四层验证）
     */
    public List<IdCardResult> extractIdCards(String text, Label label) {
        log.info("开始提取身份证号，文本长度: {}", text.length());

        // ========== 第一层：正则表达式初筛 ==========
        List<IdCardCandidate> candidates = stage1_RegexMatch(text);
        log.info("第一层（正则初筛）: 找到 {} 个候选", candidates.size());

        if (candidates.isEmpty()) {
            return Collections.emptyList();
        }

        // ========== 第二层：格式验证 ==========
        List<IdCardCandidate> formatValidated = stage2_FormatValidation(candidates);
        log.info("第二层（格式验证）: 通过 {} 个", formatValidated.size());

        if (formatValidated.isEmpty()) {
            return Collections.emptyList();
        }

        // ========== 第三层：DeepSeek 70B语义验证 ==========
        List<IdCardResult> finalResults = stage3_SemanticValidation(text, formatValidated, label);
        log.info("第三层（语义验证）: 最终确认 {} 个", finalResults.size());

        return finalResults;
    }

    /**
     * 第一层：正则表达式初筛
     * 提取所有符合18位身份证格式的数字序列
     */
    private List<IdCardCandidate> stage1_RegexMatch(String text) {
        List<IdCardCandidate> candidates = new ArrayList<>();

        // 正则1：标准18位（17位数字 + 数字或X）
        // 严格要求：
        // - (19|20) 确保年份是19xx或20xx
        // - (0[1-9]|1[0-2]) 确保月份01-12
        // - (0[1-9]|[12]\d|3[01]) 确保日期01-31
        Pattern pattern1 = Pattern.compile(
            "(?<!\\d)[1-9]\\d{5}(19|20)\\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\\d|3[01])\\d{3}[0-9Xx](?!\\d)"
        );

        Matcher matcher1 = pattern1.matcher(text);
        while (matcher1.find()) {
            candidates.add(new IdCardCandidate(matcher1.group(), matcher1.start()));
        }

        // 正则2：带分隔符的格式（如：110101 19900101 011234 或 110101-19900101-011234）
        Pattern pattern2 = Pattern.compile(
            "[1-9]\\d{5}[\\s\\-\\.]?[1-2]\\d{3}[01]\\d[0-3]\\d[\\s\\-\\.]?\\d{4}[\\s\\-\\.]?[0-9Xx]"
        );

        Matcher matcher2 = pattern2.matcher(text);
        while (matcher2.find()) {
            String raw = matcher2.group();
            // 移除分隔符并标准化
            String cleaned = raw.replaceAll("[\\s\\-\\.]", "").toUpperCase();

            // 验证清理后的格式
            if (cleaned.matches("[1-9]\\d{5}(19|20)\\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\\d|3[01])\\d{3}[0-9X]")) {
                candidates.add(new IdCardCandidate(cleaned, matcher2.start()));
            }
        }

        // 去重（保留第一次出现的位置）
        Map<String, IdCardCandidate> uniqueMap = new LinkedHashMap<>();
        for (IdCardCandidate candidate : candidates) {
            uniqueMap.putIfAbsent(candidate.getValue(), candidate);
        }

        return new ArrayList<>(uniqueMap.values());
    }

    /**
     * 第二层：格式验证
     * 验证地区码、出生日期、校验位
     */
    private List<IdCardCandidate> stage2_FormatValidation(List<IdCardCandidate> candidates) {
        List<IdCardCandidate> validated = new ArrayList<>();

        for (IdCardCandidate candidate : candidates) {
            String idCard = candidate.getValue();

            try {
                // 1. 验证地区码（前6位）
                String areaCode = idCard.substring(0, 6);
                if (!areaCodeService.isValidAreaCode(areaCode)) {
                    log.debug("地区码无效: {}", areaCode);
                    continue;
                }

                // 2. 验证出生日期（第7-14位）
                String birthDateStr = idCard.substring(6, 14);
                LocalDate birthDate = validateBirthDate(birthDateStr);
                if (birthDate == null) {
                    log.debug("出生日期无效: {}", birthDateStr);
                    continue;
                }

                // 3. 验证校验位（第18位）
                char lastChar = idCard.charAt(17);
                char calculatedCheckDigit = calculateChecksum(idCard.substring(0, 17));
                if (lastChar != calculatedCheckDigit) {
                    log.debug("校验位错误: {} (应为 {})", lastChar, calculatedCheckDigit);
                    continue;
                }

                // 所有验证通过
                candidate.setAreaCode(areaCode);
                candidate.setBirthDate(birthDate);
                candidate.setProvince(areaCodeService.getProvince(areaCode));
                candidate.setCity(areaCodeService.getCity(areaCode));
                candidate.setPassedFormatValidation(true);

                validated.add(candidate);

            } catch (Exception e) {
                log.debug("格式验证异常: {}, 错误: {}", idCard, e.getMessage());
            }
        }

        return validated;
    }

    /**
     * 验证出生日期
     */
    private LocalDate validateBirthDate(String dateStr) {
        try {
            LocalDate date = LocalDate.parse(dateStr, DateTimeFormatter.ofPattern("yyyyMMdd"));

            // 检查日期合理性
            LocalDate now = LocalDate.now();
            LocalDate minDate = LocalDate.of(1900, 1, 1);

            if (date.isAfter(now) || date.isBefore(minDate)) {
                return null;
            }

            return date;
        } catch (DateTimeParseException e) {
            return null;
        }
    }

    /**
     * 计算身份证校验位（ISO 7064:1983.MOD 11-2算法）
     */
    private char calculateChecksum(String id17) {
        // 权重因子
        int[] weights = {7, 9, 10, 5, 8, 4, 2, 1, 6, 3, 7, 9, 10, 5, 8, 4, 2};
        // 校验码对应值
        char[] checkChars = {'1', '0', 'X', '9', '8', '7', '6', '5', '4', '3', '2'};

        int sum = 0;
        for (int i = 0; i < 17; i++) {
            sum += (id17.charAt(i) - '0') * weights[i];
        }

        return checkChars[sum % 11];
    }

    /**
     * 第三层：DeepSeek 70B语义验证
     * 使用大模型判断上下文是否明确标识为身份证号
     */
    private List<IdCardResult> stage3_SemanticValidation(String text, List<IdCardCandidate> candidates, Label label) {
        List<IdCardResult> results = new ArrayList<>();

        for (IdCardCandidate candidate : candidates) {
            // 提取上下文（身份证号前后各50个字符）
            String context = extractContext(text, candidate);

            // 调用DeepSeek 70B进行语义验证
            SemanticValidationResult svr = validateWithDeepSeek70B(context, candidate.getValue(), label);

            if (svr.isValidIdCard()) {
                // 确认为身份证号
                results.add(IdCardResult.builder()
                    .idCard(candidate.getValue())
                    .province(candidate.getProvince())
                    .city(candidate.getCity())
                    .birthDate(candidate.getBirthDate())
                    .confidence(svr.getConfidence())
                    .reasoning(svr.getReasoning())
                    .context(context)
                    .validationSteps(Arrays.asList(
                        "✅ 正则匹配通过",
                        "✅ 地区码验证通过: " + candidate.getAreaCode(),
                        "✅ 日期验证通过: " + candidate.getBirthDate(),
                        "✅ 校验位验证通过",
                        "✅ DeepSeek 70B语义验证通过"
                    ))
                    .build());
            } else {
                log.info("排除疑似身份证号: {}, 原因: {}", candidate.getValue(), svr.getReasoning());
            }
        }

        return results;
    }

    /**
     * 提取上下文（前后各50个字符）
     */
    private String extractContext(String text, IdCardCandidate candidate) {
        int pos = candidate.getPosition();
        int start = Math.max(0, pos - 50);
        int end = Math.min(text.length(), pos + candidate.getValue().length() + 50);

        return text.substring(start, end);
    }

    /**
     * 使用DeepSeek 70B进行语义验证
     */
    private SemanticValidationResult validateWithDeepSeek70B(String context, String idCard, Label label) {
        try {
            // 构建提示词
            String prompt = buildSemanticPrompt(context, idCard);

            // 调用DeepSeek 70B（低温度确保一致性）
            Map<String, Object> config = new HashMap<>();
            config.put("temperature", 0.1);
            config.put("maxTokens", 300);

            String response = deepSeekService.chat(prompt, config);

            // 解析JSON响应
            return parseSemanticResponse(response);

        } catch (Exception e) {
            log.error("DeepSeek语义验证失败: {}", e.getMessage());
            // 失败时保守策略：通过格式验证的都认为有效
            return SemanticValidationResult.builder()
                .validIdCard(true)
                .confidence(0.7f)
                .reasoning("语义验证失败，基于格式验证通过")
                .build();
        }
    }

    /**
     * 构建语义验证提示词
     */
    private String buildSemanticPrompt(String context, String idCard) {
        return String.format("""
            你是专业的信息提取专家。请判断以下文本中的数字序列是否确实为身份证号。

            【待判断的数字序列】
            %s

            【上下文文本】
            "...%s..."

            【判断标准】
            1. **明确标识**：上下文中是否有明确的关键词，如：身份证、证件号、身份证号、身份证号码、ID、身份证明、居民身份证等
            2. **格式特征**：已通过正则、地区码、日期、校验位四项验证
            3. **语义合理性**：该数字序列在上下文中被用作身份证标识

            【排除标准】
            1. 订单号、序列号、编号、流水号、单号
            2. 随机的18位数字序列
            3. 手机号、银行卡号、发票号等其他号码
            4. 明确标注为非身份证的号码

            【特别注意】
            - 如果上下文中有"身份证"相关关键词，优先判定为身份证号
            - 如果是15位数字，即使格式相似，也不应判定为18位身份证号
            - 如果不确定，但格式验证全部通过，可以倾向于判定为身份证号

            请按以下JSON格式返回结果：
            {
              "isValidIdCard": true 或 false,
              "confidence": 0-100的整数,
              "reasoning": "简要说明判断依据（不超过100字）"
            }

            请只返回JSON，不要有任何其他内容。
            """, idCard, context);
    }

    /**
     * 解析语义验证响应
     */
    private SemanticValidationResult parseSemanticResponse(String response) {
        try {
            // 提取JSON（可能包含markdown代码块）
            String jsonStr = extractJson(response);

            JsonNode json = objectMapper.readTree(jsonStr);

            return SemanticValidationResult.builder()
                .validIdCard(json.get("isValidIdCard").asBoolean())
                .confidence(json.get("confidence").asInt() / 100.0f)
                .reasoning(json.get("reasoning").asText())
                .build();

        } catch (Exception e) {
            log.warn("解析语义验证响应失败: {}, 响应: {}", e.getMessage(), response);
            // 解析失败时保守策略
            return SemanticValidationResult.builder()
                .validIdCard(true)
                .confidence(0.6f)
                .reasoning("响应解析失败，基于格式验证通过")
                .build();
        }
    }

    /**
     * 从响应中提取JSON字符串
     */
    private String extractJson(String response) {
        // 尝试提取第一个完整的JSON对象
        int start = response.indexOf("{");
        int end = response.lastIndexOf("}");

        if (start != -1 && end != -1 && end > start) {
            return response.substring(start, end + 1);
        }

        return response;
    }
}
```
