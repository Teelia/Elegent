## 二、功能测试

### 测试 1: 创建身份证号提取标签

1. 进入 **标签库** 页面
2. 点击 **新建标签**
3. 填写表单：
   - 名称: `身份证号提取`
   - 类型: 选择 **结构化号码提取**
   - 提取器类型: 选择 **身份证号提取**
   - 勾选:
     - 包含18位标准: ✅
     - 包含15位旧版: ✅
     - 包含疑似号码: ✅
   - 提取字段: `身份证号`
   - 重点列: `警情内容`
4. 点击 **保存**

1. 进入 **标签库** 页面
2. 点击 **新建标签**
3. 填写表单：

   - 类型: 选择 **结构化号码提取**
   - 提取器类型: 选择 **复合提取（多种号码）**

## 五、常见问题排查

### 问题 1: 提取结果为空

**可能原因**:
- 重点列配置不正确
- 文本中没有符合格式的号码

**解决方法**:
- 检查重点列配置是否正确
- 检查数据是否包含目标号码
- 尝试不设置重点列（使用所有列）

### 问题 2: 身份证号提取不准确

**可能原因**:
- 身份证号格式不符合标准
- 校验位验证失败

**解决方法**:
- 勾选"包含疑似号码"选项
- 检查原始数据格式

### 问题 3: 任务执行失败

**可能原因**:
- 数据库迁移未执行
- 后端服务未重启

**解决方法**:
- 确认数据库迁移已执行
- 重启后端服务
- 查看后端日志

{
  "name": "身份证号提取",
