# 智能数据标注平台 V3.0 功能升级优化方案

> **项目名称**：大模型自动化数据标签
> **版本**：V3.0
> **编写日期**：2025-12-29
> **技术栈**：Vue 3 + Element Plus + Spring Boot 2.7 + MySQL 8.0 + Redis

---

## 📋 需求概述

本次优化包含三个核心方向：

### 1️⃣ 提示词模板管理化
- 将硬编码的提示词逻辑提取到数据库
- 支持在页面上创建、编辑、测试提示词模板
- 支持模板版本管理和 A/B 测试

### 2️⃣ 标签规则系统增强
- 预置内置提取规则（手机号、身份证号、邮箱等正则）
- 支持"大模型判断"和"规则匹配"双模式
- 支持规则的逻辑组合（AND/OR/NOT）
- 规则优先级控制

### 3️⃣ 外接数据源支持
- 支持 Oracle 数据库导入
- 支持 PostgreSQL 数据库导入
- 统一数据源配置管理
- SQL 查询结果直接作为数据集

**约束条件**：
- ✅ 不增加额外中间件
- ✅ 数据库使用 MySQL
- ✅ 缓存使用 Redis
- ✅ 保持现有架构稳定

```java
@Service
public class PromptTemplateService {

    @Autowired
    private PromptTemplateRepository promptTemplateRepository;

    @Autowired
    private DeepSeekService deepSeekService;

    /**
     * 创建提示词模板
     */
    @Transactional
    public PromptTemplate createTemplate(CreatePromptTemplateRequest request) {
        // 1. 验证模板格式（变量占位符语法检查）
        validateTemplate(request.getUserPromptTemplate());

        // 2. 解析变量定义
        List<TemplateVariable> variables = parseVariables(request);

        // 3. 创建模板
        PromptTemplate template = PromptTemplate.builder()
            .userId(request.getUserId())
            .name(request.getName())
            .version(getNextVersion(request.getUserId(), request.getName()))
            .category(request.getCategory())
            .systemPrompt(request.getSystemPrompt())
            .userPromptTemplate(request.getUserPromptTemplate())
            .modelConfigId(request.getModelConfigId())
            .temperature(request.getTemperature())
            .maxTokens(request.getMaxTokens())
            .description(request.getDescription())
            .variables(objectMapper.writeValueAsString(variables))
            .build();

        return promptTemplateRepository.save(template);
    }

    /**
     * 更新模板（创建新版本）
     */
    @Transactional
    public PromptTemplate updateTemplate(Integer templateId, UpdatePromptTemplateRequest request) {
        PromptTemplate oldTemplate = getTemplateById(templateId);

        PromptTemplate newTemplate = PromptTemplate.builder()
            .userId(oldTemplate.getUserId())
            .name(oldTemplate.getName())
            .version(oldTemplate.getVersion() + 1)
            .category(request.getCategory())
            .systemPrompt(request.getSystemPrompt())
            .userPromptTemplate(request.getUserPromptTemplate())
            .modelConfigId(request.getModelConfigId())
            .temperature(request.getTemperature())
            .maxTokens(request.getMaxTokens())
            .description(request.getDescription())
            .build();

        return promptTemplateRepository.save(newTemplate);
    }

    /**
     * 构建最终提示词（变量替换）
     */
    public String buildPrompt(Integer templateId, Map<String, Object> params) {
        PromptTemplate template = getTemplateById(templateId);

        String userPrompt = template.getUserPromptTemplate();

        // 替换变量占位符
        for (Map.Entry<String, Object> entry : params.entrySet()) {
            String placeholder = "${" + entry.getKey() + "}";
            userPrompt = userPrompt.replace(placeholder, String.valueOf(entry.getValue()));
        }

        return template.getSystemPrompt() + "\n\n" + userPrompt;
    }

    /**
     * 测试模板
     */
    public TestResult testTemplate(Integer templateId, Map<String, Object> testData) {
        PromptTemplate template = getTemplateById(templateId);

        // 构建提示词
        String prompt = buildPrompt(templateId, testData);

        // 调用 DeepSeek API
        String result = deepSeekService.chat(prompt, template);

        // 返回测试结果
        return TestResult.builder()
            .prompt(prompt)
            .response(result)
            .tokens(calculateTokens(prompt + result))
            .build();
    }

    /**
     * 获取默认模板
     */
    public PromptTemplate getDefaultTemplate(String category) {
        return promptTemplateRepository.findByCategoryAndIsDefault(category, true)
            .orElseThrow(() -> new RuntimeException("未找到默认模板: " + category));
    }
}
```

### 2.3 规则引擎实现

#### 核心服务类：`RuleEngineService.java`

```java
@Service
public class RuleEngineService {

    @Autowired
    private BuiltinRuleRepository builtinRuleRepository;

    @Autowired
    private DeepSeekService deepSeekService;

    /**
     * 执行标签规则（支持所有模式）
     */
    public LabelResult executeRule(Label label, Map<String, Object> rowData) {
        String ruleType = label.getRuleType();

        switch (ruleType) {
            case "rule":
                return executeRuleOnly(label, rowData);
            case "llm":
                return executeLLMOnly(label, rowData);
            case "hybrid":
                return executeHybrid(label, rowData);
            default:
                throw new UnsupportedOperationException("不支持的规则类型: " + ruleType);
        }
    }

    /**
     * 纯规则匹配模式
     */
    private LabelResult executeRuleOnly(Label label, Map<String, Object> rowData) {
        JSONObject ruleConfig = JSON.parseObject(label.getRuleConfig());
        JSONArray rules = ruleConfig.getJSONArray("rules");
        String logic = ruleConfig.getString("logic");

        List<Boolean> results = new ArrayList<>();

        for (int i = 0; i < rules.size(); i++) {
            JSONObject ruleObj = rules.getJSONObject(i);
            Integer builtinRuleId = ruleObj.getInteger("builtinRuleId");
            String targetColumn = ruleObj.getString("targetColumn");

            BuiltinRule builtinRule = builtinRuleRepository.findById(builtinRuleId)
                .orElseThrow(() -> new RuntimeException("规则不存在: " + builtinRuleId));

            boolean matched = matchRule(builtinRule, rowData, targetColumn);
            results.add(matched);
        }

        boolean finalResult = "AND".equals(logic)
            ? results.stream().allMatch(r -> r)
            : results.stream().anyMatch(r -> r);

        return LabelResult.builder()
            .match(finalResult)
            .method("rule")
            .confidence(finalResult ? 100 : 0)
            .build();
    }

    /**
     * 执行单个规则匹配
     */
    private boolean matchRule(BuiltinRule rule, Map<String, Object> rowData, String targetColumn) {
        String ruleType = rule.getRuleType();
        Object targetValue = rowData.get(targetColumn);

        if (targetValue == null) {
            return false;
        }

        String targetStr = String.valueOf(targetValue);

        if ("regex".equals(ruleType)) {
            Pattern pattern = Pattern.compile(rule.getPattern());
            Matcher matcher = pattern.matcher(targetStr);
            return matcher.find();
        } else if ("conditional".equals(ruleType)) {
            JSONObject logic = JSON.parseObject(rule.getExtractionLogic());
            return evaluateConditional(logic, targetStr);
        }

        return false;
    }

    /**
     * 混合模式
     */
    private LabelResult executeHybrid(Label label, Map<String, Object> rowData) {
        JSONObject ruleConfig = JSON.parseObject(label.getRuleConfig());
        JSONObject firstStage = ruleConfig.getJSONObject("firstStage");
        JSONObject secondStage = ruleConfig.getJSONObject("secondStage");
        String trigger = ruleConfig.getString("trigger");

        // 第一阶段：规则匹配
        LabelResult firstResult = executeRuleOnly(label, rowData);

        // 判断是否需要执行第二阶段
        boolean needSecondStage = false;
        if ("always".equals(trigger)) {
            needSecondStage = true;
        } else if ("first_pass".equals(trigger)) {
            needSecondStage = firstResult.isMatch();
        } else if ("first_fail".equals(trigger)) {
            needSecondStage = !firstResult.isMatch();
        }

        LabelResult finalResult;
        if (needSecondStage) {
            // 第二阶段：大模型确认
            LabelResult secondResult = deepSeekService.classify(label, rowData);
            finalResult = secondResult;
            finalResult.setFirstStageResult(firstResult);
        } else {
            finalResult = firstResult;
        }

        return finalResult;
    }
}
```

```java
            String dbVersion = metaData.getDatabaseProductVersion();

            ds.setConnectionStatus("success");
            ds.setLastConnectedAt(LocalDateTime.now());
            ds.setErrorMessage(null);
            dataSourceRepository.save(ds);

            return ConnectionTestResult.builder()
                .success(true)
                .databaseVersion(dbVersion)
                .build();

        } catch (Exception e) {
            ds.setConnectionStatus("failed");
            ds.setErrorMessage(e.getMessage());
            dataSourceRepository.save(ds);

            return ConnectionTestResult.builder()
                .success(false)
                .errorMessage(e.getMessage())
                .build();
        } finally {
            if (conn != null) {
                try { conn.close(); } catch (Exception e) {}
            }
        }
    }

    /**
     * 获取数据源的表列表
     */
    public List<TableInfo> listTables(Integer dataSourceId) {
        DataSource ds = dataSourceRepository.findById(dataSourceId)
            .orElseThrow(() -> new RuntimeException("数据源不存在"));

        javax.sql.DataSource dataSource = DataSourceConfig.createDataSource(ds);
        Connection conn = null;

        try {
            conn = dataSource.getConnection();
            DatabaseMetaData metaData = conn.getMetaData();

            ResultSet rs = metaData.getTables(
                ds.getDatabaseName(),
                null,
                "%",
                new String[]{"TABLE"}
            );

            List<TableInfo> tables = new ArrayList<>();
            while (rs.next()) {
                tables.add(TableInfo.builder()
                    .tableName(rs.getString("TABLE_NAME"))
                    .tableType(rs.getString("TABLE_TYPE"))
                    .remarks(rs.getString("REMARKS"))
                    .build());
            }

            return tables;

        } catch (Exception e) {
            throw new RuntimeException("获取表列表失败", e);
        } finally {
            if (conn != null) {
                try { conn.close(); } catch (Exception e) {}
            }
        }
    }

    /**
     * 执行查询并导入为数据集
     */
    @Transactional
    public DatasetImportResult importQueryAsDataset(Integer userId, ImportQueryRequest request) {
        DataSource ds = dataSourceRepository.findById(request.getDataSourceId())
            .orElseThrow(() -> new RuntimeException("数据源不存在"));

        javax.sql.DataSource dataSource = DataSourceConfig.createDataSource(ds);
        Connection conn = null;
        PreparedStatement stmt = null;
        ResultSet rs = null;

        try {
            long startTime = System.currentTimeMillis();

            conn = dataSource.getConnection();
            stmt = conn.prepareStatement(request.getSqlQuery());
            rs = stmt.executeQuery();

            // 获取结果集元数据
            ResultSetMetaData metaData = rs.getMetaData();
            int columnCount = metaData.getColumnCount();
            List<String> columnNames = new ArrayList<>();
            for (int i = 1; i <= columnCount; i++) {
                columnNames.add(metaData.getColumnLabel(i));
            }

            // 创建数据集
            Dataset dataset = Dataset.builder()
                .userId(userId)
                .name(request.getDatasetName())
                .sourceType("database")
                .sourceId(query.getId())
                .totalRows(0)
                .build();
            dataset = datasetRepository.save(dataset);

            // 读取数据并保存
            int rowCount = 0;
            List<DataRow> dataRowsBatch = new ArrayList<>();
            final int BATCH_SIZE = 1000;

            while (rs.next()) {
                Map<String, Object> rowData = new LinkedHashMap<>();
                for (int i = 1; i <= columnCount; i++) {
                    String columnName = columnNames.get(i - 1);
                    Object value = rs.getObject(i);
                    rowData.put(columnName, value);
                }

                DataRow dataRow = DataRow.builder()
                    .datasetId(dataset.getId())
                    .rowIndex(rowCount)
                    .originalData(objectMapper.writeValueAsString(rowData))
                    .processingStatus("pending")
                    .build();

                dataRowsBatch.add(dataRow);
                rowCount++;

                if (dataRowsBatch.size() >= BATCH_SIZE) {
                    dataRowRepository.saveAll(dataRowsBatch);
                    dataRowsBatch.clear();
                }
            }

            if (!dataRowsBatch.isEmpty()) {
                dataRowRepository.saveAll(dataRowsBatch);
            }

            dataset.setTotalRows(rowCount);
            datasetRepository.save(dataset);

            return DatasetImportResult.builder()
                .success(true)
                .datasetId(dataset.getId())
                .rowCount(rowCount)
                .columnNames(columnNames)
                .executionTime((int)(System.currentTimeMillis() - startTime))
                .build();

        } catch (Exception e) {
            throw new RuntimeException("查询执行失败", e);
        } finally {
            // 关闭资源
        }
    }
}
```

## ⚠️ 六、注意事项与风险

### 1. Oracle 驱动授权问题
- Oracle JDBC 驱动需要手动下载和安装
- 部署时需要将 `ojdbc8.jar` 放入项目的 `lib` 目录

### 2. SQL注入防护
- 必须使用参数化查询（PreparedStatement）
- 禁止用户拼接的SQL直接执行

### 3. 数据源密码安全
- 密码必须加密存储（建议使用AES）
- 日志中脱敏敏感信息

### 4. 大数据查询性能
- 查询结果集必须分页或流式处理
- 避免全表扫描
- 设置查询超时时间
