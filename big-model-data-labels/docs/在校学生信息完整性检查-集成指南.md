## 二、代码变更说明

### 2.1 新增文件

#### StudentInfoValidator.java
**路径**: `backend/src/main/java/com/datalabeling/util/StudentInfoValidator.java`

**功能**:
- 辍学学生检测
- 学校简称识别（如"北师大" → "北京师范大学"）
- 学校全称验证（检查是否缺少省/市前缀）
- 培训机构检测
- 6项信息完整性检查
- 年龄推断（6-18岁人员推断为在校生）

**使用方式**:
```java
@Autowired
private StudentInfoValidator studentInfoValidator;

public void checkStudentInfo(String result, String text) {
    StudentInfoValidator.ValidationResult validationResult =
        studentInfoValidator.validate(result, text);

    if (!validationResult.isValid()) {
        log.error("验证失败: {}", validationResult.getMessage());
    }
}
```

1. **系统启动时**：
   - `BuiltinGlobalLabelsInitializer` 会自动更新"在校学生信息完整性检查"标签的定义和提示词

## 六、故障排查

### 6.1 常见问题

#### 问题1：优化未生效

**现象**: 重新分析后仍然出现相同错误

**排查步骤**:
1. 检查 `system_prompt` 表中的 `student_info_enhancement_v1` 记录是否已更新
2. 检查 `label` 表中的描述是否已更新
3. 重启应用

#### 问题2：StudentInfoValidator 未被调用

**现象**: 日志中没有"检测到在校学生标签"的记录

**排查步骤**:
1. 确认标签名称包含"在校学生"
2. 检查 `PostProcessValidator.validate()` 的调用是否传入了 `labelName` 参数

#### 问题3：学校简称未被识别

**现象**: "北师大"等简称仍然通过验证

**排查步骤**:
1. 检查 `StudentInfoValidator.SCHOOL_ABBREVIATIONS` 是否包含该简称
2. 如需添加，修改 `SCHOOL_ABBREVIATIONS` 列表
